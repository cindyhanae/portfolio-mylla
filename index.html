Stick Together with Super Bonder (2020)

What I did:

Designed the gameplay and art style. Drew art using Illustrator tool. Helped programming the game and debug issues.


About the game:

The game was developed on Global Game Jam 2020, in which the theme was "Repair". I worked with my friend as a duo. Since we both love cute art style, we decided to pick a smaller and more manageable game scope with a single mechanic, but with a cute art style. We were both set on making a co-op multiplayer game so it could be family-friendly and be enjoyed by the younger public.


Link: Global Game Jam game page

❮ ❯

Supermarket Battle (2019)

What I did:

Helped with game idea and design. Responsible for the gameplay programming.


About the game:

Our small team of 3 was selected to participate in the BGS Jam 2019, a game jam that happened inside Brazil Game Show event.

We were supposed to make a game with the "find and seek" mechanic. That was the theme. Our objective was to make a fun game with a competitive twist. After a long brainstorming session, we decided to make a local PvP multiplayer game in a supermarket where you must collect the most items as possible.

❮ ❯

Fluffy Skirmish (2018)

What I did:

Designed game mechanic. Drew 2D art. Responsible for the gameplay and multiplayer programming.


About the game:

This game was my team's final project for college. Since we had a 1-year period to develop the game project, we decided to challenge ourselves and make a multiplayer game. We first decided to make a cute battle royale game, since all of us were into games of the same genre at the time.

After discussing a lot and improving the game design, we agreed to make it a FFA game with a "battle-royale-ics" mechanic where the map could also kill the player.


Link: itch.io game page
